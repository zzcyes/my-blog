# 前端面试规划

![interview-001.png](../../images/interview-001.png)

**技术栈掌握程度:**

**「了解」** ⭐

- 会使用/用过

- 了解其思想没有深入了解过

- 简单阅读过该技术的文档或者相关文章，知道这么一个东西

**「熟悉」** ⭐⭐

- 有大量实践

- 有深入了解过原理，甚至阅读过源码

- 遇到复杂问题能够快速想出解决方案

- 他人提出问题时，可以快速想到答案或者思路

**「精通」** ⭐⭐⭐

- 深入理解原理

- 大量复杂场景实践经验

- 大量相关源码经验

- 可以指导他人
