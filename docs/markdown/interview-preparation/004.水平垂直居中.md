## 一、水平垂直居中

### flex

定宽高、不定宽高

![css-center-001.png](../../images/css-center-001.png)

### 方法一

```css
.parent {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

### 方法二

```css
.parent {
  display: flex;
}

.child {
  align-self: center;
  margin: 0 auto;
}
```
