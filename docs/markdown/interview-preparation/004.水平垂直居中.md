### flex

![css-center-001.png](../../images/css-center-001.png)

```html
<div class="parent">
  <div class="child">box</div>
</div>
```

```css
.parent {
  width: 500px;
  height: 500px;
  display: flex;
  justify-content: center;
  align-items: center;
  background-color: gray;
}

.parent > .child {
  width: 200px;
  height: 200px;
  background-color: pink;
}
```
